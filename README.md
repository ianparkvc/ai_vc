AI_VC
AI-Powered VC Investment Committee Simulation

여러분의 피치덱, 20명의 VC가 심사한다면?

AI_VC 스타트업 피치덱을 20명의 실존 VC·사상가 페르소나로 분석하는 AI 시뮬레이션 프로젝트입니다. 각 페르소나가 고유한 투자 철학과 프레임워크로 독립 평가하고, 선별된 3인이 5라운드 IC(Investment Committee) 토론을 벌입니다.

How It Works
피치덱 제출 → 20인 독립 평가 → 3인 패널 선정 → 5라운드 IC 토론 → 최종 리포트
4-Stage Analysis Pipeline:

Independent Review — 20명의 페르소나가 각자의 프레임워크로 독립 평가 (Invest / Pass / Dig Deeper)
Panel Selection — Bull, Bear, Wild Card 3인 패널 구성
5-Round IC Debate — 실제 투자위원회처럼 공격·방어·합의 과정 시뮬레이션
Final Report — 종합 판정 + 액션 플랜 도출
